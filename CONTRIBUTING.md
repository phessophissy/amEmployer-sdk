# Contributing

Thanks for helping improve the amEmployer SDK.

## Development

Install dependencies:

```bash
npm install
```

Run the TypeScript check:

```bash
npm run build:check
```

Build the package:

```bash
npm run build
```

## Pull Requests

- Keep changes focused and reviewable.
- Update `README.md` or examples when changing public SDK behavior.
- Run `npm run build:check` and `npm run build` before opening a PR.
- Include generated `dist` updates when source changes affect package output.
